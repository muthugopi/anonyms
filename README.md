# Anonyms for RIT

A secure and anonymous confession platform tailored for college students to share thoughts, experiences, and feedback.

---

## Features

* **User Authentication:** Secure signup and login system restricted to campus domains.
* **Role-Based Access Control:** Distinct workflows for standard students and content moderators.
* **Anonymous Posting:** Complete encryption or omission of user identifiers on public-facing confessions.
* **Admin Verification:** Dedicated dashboard for administrators to review, approve, or reject submissions before they go live.
* **Post Management:** Content deletion capabilities for authors and authorized moderators.

---

## Tech Stack

* **Backend:** Django (Python)
* **Database:** MySQL
* **Frontend:** Bootstrap 5, HTML5, CSS3, JavaScript (ES6)
